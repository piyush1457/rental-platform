# Rental Platform

This is a rental platform web application.

## Live Demo

[View the live site](https://rental-platform-4srx-git-main-piyush1457s-projects.vercel.app/)

## Project Structure

- `client/` — Frontend React application
- `server/` — Backend Node.js/Express API

## Features
- Browse products for rent
- Add items to cart
- Request rentals
- Track orders
- User authentication
