This is a rental platform web application.

## Project Structure

- `client/` — Frontend React application
- `server/` — Backend Node.js/Express API

## Features

## Demonstration

The Rental Platform allows users to easily rent products online:

1. Browse available products on the homepage.
2. Select items and add them to your cart.
3. Choose rental duration and review your order.
4. Sign up or log in to place a rental request.
5. Track your order status and manage your profile.

The platform streamlines the rental process, making it simple to find, request, and manage rentals from a single dashboard.

## How to Start the Project

### Prerequisites
- Node.js and npm installed
- MongoDB running locally or a MongoDB URI

### 1. Clone the repository
```
git clone <repo-url>
cd rental-platform
```

### 2. Start the Backend (Server)
1. Navigate to the server folder:
	```
	cd server
	```
2. Install dependencies:
	```
	npm install
	```
3. Create a `.env` file with your MongoDB URI:
	```
	MONGO_URI=mongodb://localhost:27017/rentalPlatform
	```
4. Start the server:
	```
	node index.js
	```

### 3. Start the Frontend (Client)
1. Open a new terminal and navigate to the client folder:
	```
	cd client
	```
2. Install dependencies:
	```
	npm install
	```
3. Start the React app:
	```
	npm start
	```
4. Visit [http://localhost:3000](http://localhost:3000) in your browser.
